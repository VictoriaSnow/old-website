Colophon

I hand-coded this site in HTML5, CSS3, and Sass. The icons are built as SVG format. I used Coda 2.5 as text editor. The font utilized is Brandon Grotesque, a clean and cool sans serif type from Adobe Typekit. The map is taken from the fantastic project Map Glyphs, the ultimate CSS map font. Big thanks to Dmitry Semenov for Magnific Popup and Kyle Foster for Freemason and Rift, two amazing jQuery plugins. Thanks for reading :)
